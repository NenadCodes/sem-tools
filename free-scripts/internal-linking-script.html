10 SEO Workflow Tips

Introduction

This tool requires some initial set-up. Basically, you must have five columns in a "csv" file, in this order and with these exact names "URL", "Title", "Link From", "Link To", and "Anchor". Then you upload your file into the Google Collab file I am sharing with you. I'll list the steps below.

1. Crawl the Webiste

Use Screaming Frog or a homemade crawler/scraper to get the internal links from the website you want to analyze.

2. Organize Your Data

You need to organize your data in these exact columns "URL", "Title", "Link From", "Link To", "Anchor". Once you've done that, export the data to a "csv" file.

3. Upload the File to the Script

Upload the "csv" file to the script, specify the path to your file and run the script.

4. Plan New Links

Plan your links. Click on the first node and then on the second node to create new internal links.

5. Export and Implement

Once you are done, click on the "Export Links" button and you will receive a "csv" file with the new internal links. Then you can go ahead and implement them on your website.
