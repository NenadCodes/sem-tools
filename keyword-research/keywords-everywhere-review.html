Keywords Everywhere

This tool allows you to do SEO, content marketing, competitor research, PPC and social media marketing right as you browse the web. It turns your browser into an SEM tool.

How to Use Keywords Everywhere

  1. Visit the Keywords Everywhere Website:

    Go to the Keywords Everywhere website and choose to install the browser add-on for either Chrome or Firefox.

  2. Install the Browser Extension:

    Click the appropriate button for your browser, which will take you to the extension page. Install the extension, and you will see the Keywords Everywhere icon appear in the top-right corner of your browser.

  3. Sign Up for an API Key:

    Follow the link on the website to sign up for an API key. Enter your email address, and you will receive an email containing your API key.

  4. Activate Your API Key:

    Copy the API key from the email, open the Keywords Everywhere extension, and paste the key into the settings under API settings. Click "validate" to activate it.

  5. Purchase a Subscription Plan:

    Select a subscription plan that provides credits needed to retrieve keyword data. Each credit allows access to data such as search volume and competition for one keyword.

  6. Set Up and Start Using the Tool:

    Once you've set everything up, you are ready to use Keywords Everywhere to gather keyword data.

Features

Accurate monthly search volume, CPC, competition, and 12-month trend data for keywords.

Monthly traffic estimates and top 10,000 keywords for any URL or domain.

Domain Authority (DA), DA Trend, Spam Score, Referring Domains, Backlinks.

SEO Difficulty, On-Page Difficulty, Off-Page Difficulty, Brand Queries.

Search insights, SERP metrics, video insights, YouTube tags.

Historical volume data for Google and YouTube dating back to 2004.

Import keywords to view search volume, CPC, and competition.

Obtain historical volume data in bulk.

Related keywords, PASF keywords, long-tail keywords.

Analyze keyword usage and density for any URL.

Identify keywords competitors rank for, but you don’t.

Support for Google Search, Search Console, Analytics, Keyword Planner, and Trends.

Analyze data from Amazon, eBay, and Etsy.

Google, YouTube, Bing, DuckDuckGo.

Pros and Cons

    Pros:

    • Smooth interface and excellent user experience
    • Customizable and easy to use
    • Provides essential keyword data
    • Usable globally with an API key, even on different devices
    • Compatible with over ten search engines and platforms, including popular marketplaces
    • Very affordable, with 100,000 credits costing just $10, which can last several months with careful use

    Cons:

    • Easy to forget it's turned on, which can lead to accidental credit usage
    • Provides less data compared to some competitors, requiring alternatives for more detailed information
